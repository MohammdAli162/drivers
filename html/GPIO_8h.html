Description : Input sense configuration for pin n , this will determine the interrupt event sensed on the pin .

Enumerator
SENSE_BOTH_EDGES 

The pin will get event triggered by any edge in the input.

SENSE_RISING_EDGE 

The pin will get event triggered by a rising edge in the input.

SENSE_FALLING_EDGE 

The pin will get event triggered by a falling edge in the input.

SENSE_LEVEL 

The pin will get event triggered by any level change in the input.

Digital input is disabled completely on this pin

◆ opcconfig

enum opcconfig

Description : Specifies the output/pull configuration for port pins when output and input as such output_input.

Enumerator
TOTEMPOLE_Z 

TOTEMPOLE_Z : OUTPUT is TOTEMPOLE , when input it is in Z state.

TOTEMPOLE_BUSKEEPER 

TOTEMPOLE_Z : OUTPUT is TOTEMPOLE , when input it is in Z state.

TOTEMPOLE_PULLDOWN 

TOTEMPOLE_BUSKEEPER : OUTPUT is TOTEMPOLE , When input it is in buskeeper mode which preserves the last input on the pin.

TOTEMPOLE_PULLUP 

TOTEMPOLE_PULLUP : OUTPUT is TOTEMPOLE , When input it is pulled up by an active resistor.

WIREDOR_Z 

WIREDOR_Z : OUTPUT is ORWIRED ( when OUTPUT is High , the pin is high . when OUTPUT is low , it is open drain(Z) ) , when input it is open drain / Z.

WIREDAND_Z 

WIREDAND_Z : OUTPUT IS ANDWIRED( when OUTPUT is low , the pin is low . when OUTPUT is high , it is Z ) , When input , its is in Z state.

WIREDOR_PULLDOWN 

WIREDOR_PULLDOWN : Same as WIREDOR_Z but instead of Z states in the output and input it is pulled low internally instead.
